# 🌐 SMK IT Nurul Hidayah Palalangon Website

Website profil resmi **SMK IT Nurul Hidayah Palalangon** yang dibuat menggunakan **HTML, CSS, dan JavaScript murni (tanpa framework)**.

🔗 **Live Demo:**
https://la-maqiyya.github.io/Web-SMK-IT-NURUL-HIDAYAH-PALALANGON/

---

## ✨ Fitur Utama

* ✅ Desain **modern islami (light & dark mode)**
* ✅ **Responsive** (desktop & mobile)
* ✅ Navbar interaktif + smooth scroll
* ✅ Modal **Login & Kontak**
* ✅ Validasi form (tanpa backend)
* ✅ Animasi scroll (AOS)
* ✅ Counter statistik (hero section)
* ✅ Filter galeri
* ✅ UI elegan + clean
